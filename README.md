We release the SheetCopilot as well as the evaluation environment in this repository.

SheetCopilot is an assistant agent that manipulate spreadsheets by following your command. It breaks new ground in human-computer interaction, opening up possibilities for enabling non-expert users to complete their mandane work on complex software (e.g. Google sheets and Excel) via a language interface.
